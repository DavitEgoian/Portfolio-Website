# TokenWise Pitch Deck

10-slide investor pitch deck for **TokenWise** — AI cost optimization platform.

## View

Open `index.html` in a browser, or run a local server:

```bash
cd tokenwise-pitch-deck
npx --yes serve -p 3456
```

Then visit http://localhost:3456

## Navigation

- **Arrow keys** or **Space** — next slide
- **Left arrow** — previous slide
- On-screen navigation buttons at the bottom

## Export to PDF

1. Open the deck in Chrome
2. Press `Ctrl+P` / `Cmd+P`
3. Set layout to **Landscape**
4. Enable **Background graphics**
5. Save as PDF

Each slide is formatted for **16:9 widescreen** (1280×720).
